# Land Registry System

NFT-based land registry on ERC-721 (OpenZeppelin). Government (Owner) mints land parcels as NFTs to KYC-verified citizens. Properties can be transferred only between KYC-verified wallets.

## Commands

```shell
npx hardhat compile
npx hardhat test
npx hardhat node
npx hardhat run scripts/deploy.ts --network localhost
npx hardhat run scripts/deploy.ts --network amoy
```
